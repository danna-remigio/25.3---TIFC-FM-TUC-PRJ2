 El Sitio del Blog – Fugglers

Este proyecto es un sitio web tipo blog creado como parte de una actividad académica.
El tema del blog está inspirado en los **Fugglers (Funny Ugly Monsters)**, juguetes conocidos
por su apariencia extraña y divertida.

El objetivo del proyecto es poner en práctica conocimientos básicos de **HTML**, **CSS**,
el uso de **GitHub** y la metodología **SCRUM**.
