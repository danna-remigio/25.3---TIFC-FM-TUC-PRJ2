Sobre este proyecto

Este blog fue creado por un equipo de estudiantes como parte de un proyecto académico enfocado en el aprendizaje de HTML y CSS. Nuestro objetivo fue desarrollar un sitio web funcional, creativo y visualmente atractivo.

Nuestro equipo

Somos un equipo que está aprendiendo desarrollo web y decidió trabajar de manera colaborativa utilizando GitHub y la metodología SCRUM. Cada integrante aportó ideas para el diseño, contenido y estructura del sitio.

¿Por qué elegimos a los Fugglers?

Elegimos a los Fugglers porque son juguetes diferentes a lo común. Su apariencia rara y divertida los hace ideales para un blog que busca llamar la atención.

Objetivo del blog

El objetivo de este blog es demostrar nuestros avances en programación web básica, aplicando buenas prácticas de estructura, navegación y estilo.
